# ProjetoFinal_FrontEnd
Projeto Final da matéria Front-End essencial do Serratec. Nesse trabalho utilizamos apenas HTML, CSS e JAVASCRIPT puro, onde tivemos que criar um campo de login e senha, com validação e gravação dos dados no LocalStorage. 

A proposta do trabalho foi consumir uma API através de um endpoint livre, e criar pelo menos um objeto através de uma classe. A validação de acesso através de um token protegendo o acesso e permanência de um usuário não logado a uma determinada página. 

Obs.: O trabalho original foi alterado apenas na imagem de fundo da página do Cliente, devido ao tamanho do video. Nesse caso foi inserido uma imagem estatica retirada a partir desse vídeo.
